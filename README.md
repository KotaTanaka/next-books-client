# next-books-spa

書籍管理SPA [Next.js]

*[バックエンドリポジトリ nest-books-api](https://github.com/KotaTanaka/nest-books-api)*

## Technology

* [Next.js](https://nextjs.org)

## Getting Started

**Install**

```
$ git clone git@github.com:KotaTanaka/next-books-spa.git
```

```
$ cd next-books-spa
$ yarn
```

**Run**

```
$ yarn dev
```

→ http://localhost:3091
